# Controle de Acesso

## V ou F
--- 
### (1)
O termo Controle de Acesso faz referência a importância que deve existir no fornecimento de acesso e uso dos dados armazenados nos Sistemas Gerenciadores de Banco de Dados (SGBD).
 
Estando os dados seguros no SGBD não existe mais a preocupação com o acesso físico ao local onde o hardware da servidora de banco de dados esteja instalado e operando

**```Resposta: Falso```**

---
### (2)
O profissional de administração do Sistema Gerenciador de Banco de Dados (SGBD) pode precisar apagar um usuário do SGBD.
 
Essa situação é comum na administração de banco de dados e a instrução SQL
     DROP USER ANA;
 apagaria o usuário ANA do SGBD.
 
Após a execução dessa instrução esse usuário NÃO teria acesso ao SGBD.

**```Resposta: Verdadeiro```**

---
